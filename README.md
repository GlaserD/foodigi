## Development

### Installing Dependencies

From within the root directory for full stack:

```sh
npm install
npm run dev
```

You can try it out on your mobile device over wifi by hitting `[your-local-ip-address:[port]]` with your mobile browser.

example:

```sh
192.168.1.5:1337
```
